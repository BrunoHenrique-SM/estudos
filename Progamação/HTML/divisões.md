Para uma código mais bem estruturado e para manutenção de forma eficiente é extremamente importante e necessário que se use os seguintes códigos [[HTML]]:

```
* id="..." ---->> Colocar um ID único no documento, para identificar com CCS e/ou JS (img, div, etc).
* class="..." ---->> Classificar uma ou mais parte no documento, para identificar o conjunto com CCS e/ou JS (img, div, etc).

<head> </head> ---->> "Cabeça" do documento, informações mais relevante como título e etc.

<body> </body> ---->> "Corpo" do documento, conteúdo do código, o que dá estrutura.

  <div> </div> ---->> Divisão no documento, para fazer divisões.
  
  <header> </header> ---->> Cabeçalho da página (título, logo, navegação).
  
  <nav> </nav> ---->> Seção de navegação ou menu.
  	
  <main> </main> ---->> Conteúdo principal do site. Onde expande o documento e sua aplicação.
  
  <article> </article> ---->> Conteúdo independente, como post ou comentário.
  
  <aside> </aside> ---->> Conteúdo relacionado mas não essencial para o principal conteúdo.
  
  <section> </section> ---->> Utilizado para demonstrar as opções, a seção.
  
  <footer> </footer> ---->> Rodapé da página, links, copyright, informações de contato, do site e etc.

```
