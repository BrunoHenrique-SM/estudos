Aqui está uma série de códigos que são usados para a escrita no [[HTML]].
```
<!-- ... -->> ---->> Colocar comentários no código, facilitando a progamação e manutenção.

<title> ... </title> ---->> Título do documento, o que aparece na aba da guia.

<h1> ... </h1> ; <h6> ... </h6> ---> Título, Subtítulo, e etc.

<p> ... </p> ---->> Parágrafo no documento.
  <strong> ... </strong> ---->> Destacar conteúdo no documento.
  
<ol> </ol> ---->> Lista numerada, para fazer lista com números.
<ul> </ul> ---->> Lista não numerada, para fazer lista sem números.
  <li> ... </li> ---->> Elemento da lista, para fazer um tópico da lista.
  
<img src="..." alt="..."/> ---->> "src" para achar a imagem que irá aparecer no documento, "alt" mensagem caso não ache.
```
