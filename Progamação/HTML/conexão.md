Para fazer a conexão de um documento com outro ou fazer link direto em um site usando a escrita [[HTML]]:
```

<link rel="..." href="..." ---->> "rel" fazer relação de um documento com outro, "href" achar o documento específico.

<script src="document.js"> </script> ---->> Estabelecer relação de um documento JS com o HTML.
  defer ---->> atrasa o carregameto do JS para garantir o funcionamento (somente após estrutura completa do HTML)

<a href="link">...</a> ---->> Fazer link direto na página em um paragrafo.


```
