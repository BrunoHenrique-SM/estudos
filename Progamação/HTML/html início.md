Para se iniciar na escrita [[HTML]] é mais fácil utilizar o comando **!** dessa forma abrindo automaticamente os comandos essenciais para a escrita, estando neles incluso:

```
<!DOCTYPE html> ---->> Usado para utilizar a ultima versão do HTML.

<meta charset="UTF-8"> ---->> Para não dar erros na leitura do documento.

<meta name="viewport" content="width=device,widht, initial-scale=1.0"> ---->> Importante para funcionar em dispositivos diferentes. (tamanho?)

<html> </html> ---->> Usado para Definir Escrita em HTML.

  <html lang="pt-BR"> ---->> Usado para Definir a Língua Usada.
```
