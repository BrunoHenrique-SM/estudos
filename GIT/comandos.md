## config
A primeira coisa que se deve fazer ao instalar o git na máquina é definir o nome de usuário e email
```
git config --global user.name "Pi Kamole"  
git config --global user.email pikame@gmail.com
```
E para conferir o nome ou email da máquina basta apenas escrever o seguinte:
```
git config --global user.name
```
## init
Caso o monitoramento do projeto ainda não tenha sido iniciado, utilize o comando:
```
git init
```

## add
Para adicionar os arquivos que serão alterados ou adicionados no [[commit]], utilize o comando:
```
git add <nome-do-arquivo>
git add . //Para adicionar todos os arquivos//
```
## commit
Para realizar as alterações é através do:
```
git commit -m mensagem do commit
```
