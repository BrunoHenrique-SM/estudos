# Array
ão listas que funcionam de forma bem parecida com Python
```
const bau =[
	"Carrinho",
	"Tenis",
	true,
	22,
	{nome: Bruno},
	[1, 2, 3]
];

```
Podemos descobrir o tamanho da lista com o [[length]], equivalente ao [[range]] em Python.

# Mudando itens
É possível mudar o item do array, apenas declarando o índice do que queremos alterar. Veja o exemplo:
```
const bairros = ["Tapera", "Ribeirão", "Pantanal"];
console.log(bairros[1]);
bairros[1] = "Joaquina";

console.log(bairros[1]);
```
