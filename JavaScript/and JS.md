# &&
Esse é o [[Operador Lógico]] [[and]] em [[JavaScript]], vai retornar **True** se ambas condições forem verdadeiras. Veja um exemplo:
```
if (nomeInformado === nomeUsuario && senhaInformada === senhaUsuario) {
	console.log("Logado com sucesso!");
} else {
	console.log("Senha ou nome de usuário incorreto");	
}
```
