# Algoritmo 
É um conjunto de instruções finitas, para realizar uma tarefa. Pode ser dividido em três partes:
- Entrada
- Processamento
- Saída
Na **entrada** são captados os das **dados** necessários, se for uma **[[função]]** matemática os dados são [variáveis](Variavéis), se for uma **receita** de bolo serão os **ingredientes** e etc.
Na parte do processamento esses dados serão utilizados e transformados por comandos. Por fim na saída os resultados serão apresentados.
